# User Intake — minimum questions

After user describes project, ask minimum follow-ups. Default: decide silently.

| Question | When to skip |
|----------|--------------|
| Brand name | If already given |
| Photos: yours or stock? | If specified |
| Main CTA goal | If obvious (subscribe/book/buy) |
| Pages: one or multi? | Default=one-page; ask only if multi-page implied |
| Anything MUST include? | Free-text, optional |

**Template:**
> 1. **Nombre marca**: ¿cómo se llama?
> 2. **Fotos**: ¿tienes o uso stock? Si tienes, sube a `{project}/assets/photos/source/`
> 3. **Objetivo**: ¿qué debe hacer el visitante? (reservar/suscribirse/contactar/comprar/leer)
> 4. **¿Algo que deba incluir?** (carta/blog/mapa). Si no, paso a diseñar.
> 5. **¿Una página o varias?** (default: una)

## Don't ask (skill decides)
- Color palette, fonts, layout, effects, tech decisions, custom cursor, contact form

## After user replies
Acknowledge 1 line → go silent → execute → notify when ready.
Defaults: no photos→Openverse, pages→one-page, CTA→infer from industry.
